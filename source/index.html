What is nvslider

nvslider is a plugin for jQuery that makes it easy to create carousels and sliders on your page. Take any array of elements, wrap them in a block element and initialize the slider with options according to your needs.

Why nvslider

nvslider is open source and free to use, it offers a bunch of features out-of-the-box, and its source code is easy to understand and extend. The project is also actively developed and updated regularly.

In action

Easy to use

Just create a set of elements for your slides, put them in a wrapper and execute the slider on the set container. This is an example with an unordered list, but any element can be used. 

...
$('.slider').nvslider({
    showItems: 3,
    slideFor: 3,
    slideshow: true,
    timer: 2000
});
...
