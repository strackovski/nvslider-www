Detailed Guide

1. What you need

jQuery

nvSlider files (jquery.nvslider.js, nvslider.css)

A collection of icons called FontAwesome (included in the download)

Transfer the files to your project and reference them in the HTML document, like so: 

<html>
    <head>
        <script type="text/javascript" src="jquery-1.11.1.js"></script>
        <script type="text/javascript" src="jquery.nvslider.js"></script>
        <link rel="stylesheet" href="nvslider.css">
    </head>
    <body>
    ...
    </body>
</html>
Instead of downloading jQuery, you can also include a link of the file hosted online:
<script src="//ajax.googleapis.com/ajax/libs/jquery/1.11.1/jquery.min.js"></script>

2. Make an unordered list and populate it with list items. 

<div class="wrap">
    <ul class="myList">
        <li>Content1</li>
        <li>Content2</li>
        <li>Content3</li>
        <li>Content4</li>
        <li>Content5</li>
    </ul>
</div>

Because li element takes the whole height and width, in order to be able to style the grid elements you must enclose the contents in a span element. Also, the themes (about which we'll talk a bit later) won't work if there are no spans.

But first we'll see what the finished look is without span elements. Feel free to jump to the part with span elements.

3. Define width for a wrapper element

CSS:

.wrap {
    width:550px;
}
Width must be defined on an element that wraps the ul, not the ul itself.

4. Call the plugin

Call the plugin on an unordered list inside your document.ready() event.

$('.myList').nvslider();

With no options object passed, the defaults take effect, namely:

orientation: landscape

how many items to show: 3

how many items to slide on arrow click: 1

height: 150px

background color: #d2d2d2

arrows background: gray with 0.8 opacity

6. Styling

Since li elements take the whole width, you must use span elements to be able to adjust the look.

<div class="wrap">
    <ul class="myList">
        <li><span>Content1</span></li>
        <li><span>Content2</span></li>
        <li><span>Content3</span></li>
        <li><span>Content4</span></li>
    </ul>
</div>

An example CSS:

.myList li span{
    background: #666;
    color:#fff;
    position: absolute;
    top:10px;
    bottom:10px;
    left:10px;
    right:10px;
    padding:5px;
    overflow: hidden;

    /*text centering*/
    line-height: 160px;
    vertical-align: middle;
    text-align: center;
    font-size: 2em;
}

Call the plugin on an unordered list and set the height to 200px.

$('.myList').nvslider({height: 200px});

7. Options

Options allow you to adjust the behavior of the plugin. If no options are provided, the defaults will take effect.

type: number
default: 3
possible options: any number

Usage 
$('ul').nvslider({showItems:6});
Details

Sets the number of items to show in a row.

Available options are:

  • showItems

  • orientation

  • rows

  • slideFor

  • containerCss

  • navCss

  • navHover

  • theme

  • height


An example of a portrait mode slider, with 2 rows of 4 items shown, sliding 2 rows at a time.
